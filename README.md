📝 React + TypeScript – Lista de Tarefas
Este projeto demonstra os principais conceitos de React com TypeScript apresentados em aula, incluindo:

Componentização
Estado comuseState
Eventos
Adereços
Dica com TypeScript
Estrutura de pastas organizadas
🚀 Como rodar o projeto
1. Instale as responsabilidades
npm install
ou
yarn
2. Rodamos o projeto em modo desenvolvimento
npm run dev
ou
yarn dev
Acesse http://localhost:5173 no navegador.

🎯 Exercício – Galeria de Imagens
Objetivo
Crie uma galeria de imagens usando os conceitos apresentados no exemplo da To-Do List.

Instruções
Crie um componenteImageCard

Adereços: src(URL da imagem), legenda(string)
Exiba a imagem com uma legenda abaixo.
Não App.tsx, crie um estado para armazenar as imagens

Dica:useState<{ src: string; legenda: string }[]>
Adicione um formulário com inputs para adicionar novas imagens

Insira o URL
Entrada para legenda
Botão que adiciona ao array de imagens
Renderize uma lista de ImageCardcom.map()
